# Changelog

## [0.4.1] - 2025-12-27
### Changed
- Updated App Icon to new "Saturn" design (from user assets).

## [0.4.0] - 2025-12-27
### Changed
- App name changed to "Cosmo".
- Theme listing layout switched to single column (one card per row).
- Optimized layout for "Perguntas Diárias" card.

### Fixed
- Fixed UI overlap issue in completion screen ("Tudo por hoje" overlapping cards).

## [0.3.0] - 2025-12-26
### Added
- New minimalist app icon (Saturn/Rings symbol).
- Daily Progress Bar in "Perguntas Diárias" hero card showing remaining questions.
- Animated completion screen with vibrant entry effects (ZoomIn, FadeInDown).

### Changed
- Refactored Flashcard UI: removed category tags, titles, and dividers for a cleaner, focused experience.
- Renamed category "Casais" to "Leveza".
- Improved "Ver outros temas" button layout and alignment.

### Fixed
- Fixed Flashcard flicker bug by implementing stable rendering keys.
- Optimized app icon for all resolutions.
